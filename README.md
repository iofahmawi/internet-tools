# 🚀 حاسبة سرعة الإنترنت

أداة ويب بسيطة وسهلة الاستخدام تساعدك على معرفة **الحد الأدنى لسرعة الإنترنت** التي تحتاجها بناءً على عدد الأجهزة المتصلة لديك ونوع استخدامها.

## 🌐 جرّب الأداة الآن

**[← اضغط هنا لتجربة حاسبة السرعة](https://iofahmawi.github.io/internet-calculator/)**

## ✨ المميزات

- 📺 دعم أنواع متعددة من الأجهزة: تلفزيون 4K، FHD، هواتف، لابتوب، كونسول، وألعاب سحابية
- ⚡ حساب فوري للسرعة المطلوبة مع احتساب نسبة ذروة الاستخدام
- 💡 توصية واضحة بالباقة المناسبة لك
- 📱 متجاوب مع جميع أحجام الشاشات (موبايل وسطح المكتب)
- 🌙 واجهة عربية بالكامل بخط Tajawal

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3 (متغيرات CSS + تدرجات لونية)
- JavaScript (Vanilla JS)
- خط [Tajawal](https://fonts.google.com/specimen/Tajawal) من Google Fonts

> يُضاف 10% إضافية على المجموع لحساب أوقات الذروة.
